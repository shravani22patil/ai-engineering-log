
# AI Engineering Log

A running log of what I'm building and learning on the path to becoming an AI Engineer — LLM application engineering, agentic AI systems, RAG pipelines, multimodal AI, and the ML fundamentals underneath all of it.

This isn't a tutorial archive. It's a working notebook — concept breakdowns in my own words, small experiments that test real questions ("does hybrid search actually beat pure vector search here?"), and daily notes tracking what's clicking and what isn't.
